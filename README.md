## `iron create`

Create a new project

```
USAGE
  $ iron create -n name

OPTIONS
  -h, --help       show CLI help
  -n, --name=name  (required) project name

EXAMPLE
  $ iron new -n projectname
```

## `iron help [COMMAND]`

display help for iron

```
USAGE
  $ iron help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
